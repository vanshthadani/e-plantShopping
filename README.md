# Paradise Nursery Shopping Application

Paradise Nursery is a React-based e-commerce shopping application for purchasing houseplants. Users can browse plants by category, view plant details, add plants to a shopping cart, and manage cart items dynamically.

## Features
- Browse a variety of houseplants grouped by categories
- View plant name, image, and price
- Add plants to the shopping cart
- Increase or decrease plant quantity in the cart
- Remove plants from the cart
- View total items and total cost in the cart
- Responsive navigation bar with cart item count

## Technologies Used
- React
- Redux Toolkit
- React Router
- JavaScript
- CSS

## Project Purpose
This project was built as part of a Coursera React course final project to demonstrate understanding of React components, state management using Redux, and building a functional shopping cart application.
